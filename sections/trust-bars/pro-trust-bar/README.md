# Pro Trust Bar (App-Free Component)

A high-performance, lightweight trust-building section designed to replace paid "Trust Badge" apps. This section focuses on speed, accessibility, and merchant flexibility.

## 🚀 Performance Benefits
- **Zero App Latency**: Unlike apps that load via external scripts, this is a native Liquid component served directly by Shopify.
- **SVG Optimized**: Includes a dedicated field for raw SVG code, ensuring icons are razor-sharp and load instantly without extra HTTP requests.
- **No Monthly Costs**: Eliminates the recurring $5-$15 subscription fee associated with typical Shopify trust badge apps.

## 🛠️ Key Features
- **Merchant Autonomy**: Fully customizable via the Shopify Theme Editor (Background colors, text colors, and icon tints).
- **Hybrid Icon Support**: Supports both high-performance SVGs and traditional image uploads (PNG/JPG).
- **Responsive Design**: Automatically handles layout wrapping for mobile devices, ensuring a clean grid regardless of screen size.
- **Scoped Styling**: Uses `{{ section.id }}` to ensure styles never "leak" or conflict with other theme elements.

## 📂 Installation
1. Create a new section file in Shopify named `pro-trust-bar.liquid`.
2. Copy the code from the `.liquid` file in this folder.
3. Save and navigate to the Theme Customizer.
4. Add the **"Pro Trust Bar"** section to any page.

## 💡 Use Case
Perfect for placement on the **Home Page** above the footer or on the **Product Page** right beneath the "Add to Cart" button to increase buyer confidence and reduce bounce rates.
