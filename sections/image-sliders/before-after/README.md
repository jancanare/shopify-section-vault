# Before/After Image Slider (Native Liquid)

A high-performance image comparison tool built with native CSS `clip-path` and CSS Variables. Replaces heavy, paid Shopify apps with a lightweight, zero-dependency solution.

## 🚀 Key Features
- **Zero-JS Dependency**: Uses only 5 lines of Vanilla JS and native browser range inputs.
- **Ultra-Fast**: No external libraries (like jQuery) required.
- **Touch-Ready**: Works perfectly on mobile and tablets out of the box.
- **Merchant Controls**: Adjustable max-width and image pickers via the Shopify Theme Editor.

## 📂 Deployment
1. Create a section called `before-after.liquid`.
2. Paste the code from this folder.
3. Add the section to any template in the Theme Customizer.
